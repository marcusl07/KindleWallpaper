**T78 ‚Äî Build Quote detail view**

**T79 ‚Äî Build Quote edit/add view**

**T80 ‚Äî Implement per-quote enable/disable toggle**
Wire enable/disable in `QuoteDetailView` to update `highlights.isEnabled` in the database via `AppState`. Disabled quotes excluded from rotation pool.

**T81 ‚Äî Implement delete quote**

**T82 ‚Äî Implement Set as Current Wallpaper from detail view**

**T83 ‚Äî Implement manual add quote**

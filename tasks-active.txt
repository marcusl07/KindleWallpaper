## Phase 13 — Quote Management

**T74 — Update wallpaper renderer for missing book title and author**
In `WallpaperGenerator.swift`, handle empty/nil `bookTitle` and `author` and omit the attribution line entirely rather than rendering blank space.

**T77 — Add filter controls to Quotes list view**
Add filters to `QuotesListView`: by book title, author, enabled/disabled books only, manually added only. Filters compose with search.

**T78 — Build Quote detail view**
Create `QuoteDetailView` showing full quote text, book, author, location, date added, last shown. Actions: Edit, Set as Current Wallpaper, Enable/Disable from rotation, Delete with confirmation.

**T79 — Build Quote edit/add view**
Create `QuoteEditView` for editing all fields (quote text, book title, author, location). All fields optional except quote text. If book title + author match existing book, link to it. Save/Cancel.

**T80 — Implement per-quote enable/disable toggle**
Wire enable/disable in `QuoteDetailView` to update `highlights.isEnabled` in the database via `AppState`. Disabled quotes excluded from rotation pool.

**T81 — Implement delete quote**
Wire delete in `QuoteDetailView` to remove highlight from database. Confirmation alert before deleting. Return to list after deletion.

**T83 — Implement manual add quote**
Wire `+` button in `QuotesListView` to open `QuoteEditView` in add mode. On save, insert new highlight with nullable `bookId`. Link to existing book if title+author match.
